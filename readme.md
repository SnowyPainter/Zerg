# 저그
현직 교사 분들의 노고에 있어 연수의 짧은 영상들을 수시로 넘기는 것은 손목에 굉장히 고단한 일입니다.  
Zerg는 이러한 문제를 해결했습니다.  
코드 라이선스는 MIT입니다.  
불필요한 인공지능 사용을 자제하였습니다. 코드의 미학은 불필요함을 덜어내는 것이지요.  
## 사용법
* 끄는 방법 : ESC키를 눌러주세요.
* 연수 화면 위치 : 화면이 중앙에 오도록이면 상관 없습니다.
